## Module 01

This module introduces the foundational concepts of Git and GitHub, focusing on version control, repository management, and workflow best practices.

It aligns with the Calbright College Network Technology Program structure and serves as the starting point for managing and documenting technical work.

<!--more-->
---

### Overview

In this module, I developed an understanding of:

- Version control concepts  
- Local vs remote repositories  
- GitHub repository structure  
- Basic Git commands and workflows  
- The importance of structured documentation  

---

### Key Concepts

- **Git** as a version control system  
- **Repositories** (local and remote)  
- **Commits, branches, and merges**  
- **Synchronization between local and GitHub environments**  
- **Workflow discipline before development**
