## Overview

This guided technical walkthrough demonstrates how to design an IPv4 subnetting scheme, configure Cisco network devices, and verify end-to-end connectivity using Cisco Packet Tracer.

## Learning Objectives

By completing this exercise you will learn how to:

- Design an IPv4 subnetting scheme
- Calculate subnet masks
- Assign network and host addresses
- Configure router interfaces
- Configure switch management interfaces
- Configure host IP settings
- Verify network connectivity using Ping

---

## Lab Environment

This exercise uses a customer network consisting of two LANs connected by a Cisco router and an ISP connection.

Students design a subnetting plan, assign addresses, configure network devices, and validate connectivity between hosts and network infrastructure. :contentReference[oaicite:0]{index=0}

## Why This Matters

Subnetting is one of the foundational skills of network administration. Designing efficient IPv4 networks improves scalability, reduces broadcast traffic, conserves address space, and prepares administrators to deploy and troubleshoot enterprise networks.
