## Module 11 - Network Topologies

In this module, I developed an understanding of how network topology design impacts performance, reliability, scalability, and security. I explored both physical and logical topologies, and how modern enterprise networks use a combination of structured switching, segmentation, and wireless design to support growing organizational needs.
<!--more-->

---

### Key Concepts

- **Network Topologies**: The physical and logical layout of a network, including how devices are interconnected and how data flows between them.

- **Physical Topology**: The actual layout of cables and devices (e.g., star, mesh, hybrid).

- **Logical Topology**: How data travels across the network, regardless of physical layout.

- **VLAN (Virtual Local Area Network)**: A method of segmenting a network into separate broadcast domains to improve performance and security.

- **Trunking (IEEE 802.1Q)**: A protocol used to carry multiple VLANs across a single link between switches.

- **Redundancy**: The use of multiple paths and connections to ensure network availability in the event of failure.

- **RSTP (Rapid Spanning Tree Protocol)**: A protocol used to prevent switching loops while allowing fast failover in redundant network designs.

- **Layer 3 Switching**: Enables routing between VLANs within the switching infrastructure to improve efficiency and reduce latency.

- **Wireless Topology Design**: The strategic placement and configuration of access points to provide reliable coverage and performance.

- **SSID (Service Set Identifier)**: The name of a wireless network used to distinguish between different networks.

- **WPA3 Enterprise**: A wireless security protocol that provides strong authentication and encryption for enterprise environments.
