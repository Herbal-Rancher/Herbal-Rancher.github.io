## Overview

This milestone assessment focuses on redesigning a growing enterprise network after expansion into a second building. The redesign improves scalability, performance, redundancy, wireless coverage, and security using structured enterprise networking practices.

---

## Assessment Focus

* VLAN segmentation and trunking
* Redundant network design
* Rapid Spanning Tree Protocol (RSTP)
* Layer 3 switching and inter-VLAN routing
* Wireless enterprise deployment
* Guest and corporate wireless separation
* Network monitoring and hardening
