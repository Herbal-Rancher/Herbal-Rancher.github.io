## Why This Matters

Configuring network infrastructure is only part of building a functioning LAN. Network administrators must also configure end devices with valid IP addressing and verify successful communication across the network. This lab combines switch configuration, host configuration, and connectivity testing into a complete introductory networking exercise. :contentReference[oaicite:0]{index=0}
