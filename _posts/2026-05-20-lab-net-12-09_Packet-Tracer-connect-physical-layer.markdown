## Before You Begin

Every network begins with the Physical Layer.

Understanding interfaces, expansion modules, and cable selection is essential before configuring IP addressing or routing.

---

## Objectives

In this walkthrough you will:

- Identify router and switch interfaces.
- Examine expansion modules.
- Install network interface modules.
- Select appropriate cable types.
- Connect network devices.
- Verify interface connectivity.
- Confirm end-to-end communication.

---

## Guided Walkthrough

### Explore Device Hardware

1. Identify router management interfaces.
2. Examine LAN and WAN interfaces.
3. Review available expansion slots.

---

### Install Network Modules

1. Power down the device.
2. Install the required interface module.
3. Restart the device.
4. Verify the new interface appears.

---

### Connect the Network

1. Select the appropriate cable type.
2. Connect routers, switches, hosts, and wireless devices.
3. Verify physical connectivity.

---

### Verify Connectivity

1. Review interface status.
2. Test wired connectivity.
3. Verify wireless connectivity.
4. Confirm end-to-end communication.

---

## Verification

Verify that:

- Required modules are installed.
- Interfaces are operational.
- Devices are physically connected.
- Wired devices communicate successfully.
- Wireless devices connect successfully.
