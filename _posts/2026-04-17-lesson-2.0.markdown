## Module 02 - Network Systems

In this module, I developed an understanding of foundational network systems, including how devices communicate, how data is transmitted across networks, and the role of both wired and wireless technologies. This included exploring network models, communication protocols, and common wireless technologies such as Wi-Fi and Bluetooth.
<!--more-->
---

### Key Concepts

- **Network Systems**: The combination of hardware, software, and protocols that enable communication between devices across a network.

- **OSI Model (Open Systems Interconnection Model)**: A seven-layer framework used to understand how data moves through a network, from physical transmission to application-level communication.

- **Protocols**: Standardized rules that define how data is transmitted and received across a network, such as Transmission Control Protocol/Internet Protocol (TCP/IP).

- **Wired Networking**: Communication using physical media such as Ethernet cables, which provide stable and high-speed connections.

- **Wireless Networking**: Communication using radio frequency signals instead of cables, allowing for mobility and flexible deployment.

- **Wi-Fi (Wireless Fidelity)**: A wireless networking technology based on IEEE 802.11 standards that enables devices to connect to local area networks without physical connections.

- **Bluetooth**: A short-range wireless technology used for device-to-device communication, commonly used for peripherals such as headsets, keyboards, and mobile devices.

- **Bandwidth**: The maximum rate at which data can be transmitted across a network connection.

- **Latency**: The delay between sending and receiving data across a network.
