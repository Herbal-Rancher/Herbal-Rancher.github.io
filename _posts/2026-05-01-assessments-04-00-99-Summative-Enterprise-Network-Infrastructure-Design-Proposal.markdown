## Overview

This summative assessment presents a comprehensive enterprise network infrastructure design proposal for a growing organization with headquarters and multiple branch offices. The proposal recommends a scalable, secure, fault-tolerant architecture that supports centralized management, future growth, and layered cybersecurity controls.

---

## Assessment Focus

* Enterprise network design
* Network topology selection
* Client-server architecture
* IPv4 and IPv6 addressing
* VLAN segmentation
* Network services
* Implementation planning
* Layered network security
* Business continuity
* Scalability and redundancy

## Reflection

This summative assessment brought together the networking concepts explored throughout Module 4 into a single enterprise design proposal. Developing a complete network architecture reinforced the importance of balancing scalability, performance, security, redundancy, and maintainability while designing solutions that support long-term organizational growth.
