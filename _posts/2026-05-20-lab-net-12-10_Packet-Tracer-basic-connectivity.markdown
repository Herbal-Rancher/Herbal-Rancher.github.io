## Before You Begin

Basic switch configuration prepares network devices for management and troubleshooting.

Assigning management IP addresses enables administrators to remotely monitor and configure switches.

---

## Objectives

In this walkthrough you will:

- Configure switch hostnames.
- Configure passwords.
- Configure management interfaces.
- Assign IPv4 addresses.
- Save the running configuration.
- Verify connectivity using Ping.

---

## Guided Walkthrough

### Configure the Switches

1. Configure the hostname.
2. Configure security passwords.
3. Configure a login banner.
4. Save the configuration.

---

### Configure IPv4 Addressing

1. Configure the management VLAN.
2. Assign the management IP address.
3. Enable the interface.
4. Verify interface status.

---

### Configure End Devices

1. Assign IPv4 addresses.
2. Verify network configuration.

---

### Test Connectivity

1. Ping between hosts.
2. Ping switch management interfaces.
3. Verify successful communication.

---

## Verification

Verify that:

- Switch configuration is complete.
- Management interfaces are operational.
- End devices are correctly addressed.
- All ping tests succeed.
