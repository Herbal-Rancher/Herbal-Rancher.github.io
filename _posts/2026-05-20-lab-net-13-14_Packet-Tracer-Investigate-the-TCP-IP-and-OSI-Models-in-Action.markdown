## Overview

This guided technical walkthrough demonstrates how data moves through the TCP/IP Protocol Suite and the OSI Model using Cisco Packet Tracer Simulation Mode.

- Packet Tracer topology

- Simulation Mode

## Learning Objectives

By completing this exercise you will learn how to:

- Observe packet encapsulation
- Compare the TCP/IP and OSI models
- Analyze HTTP communication
- Examine DNS name resolution
- Follow TCP session establishment
- Interpret Protocol Data Units (PDUs)
- Explore packet flow using Simulation Mode

---

## Lab Environment

This exercise uses Cisco Packet Tracer Simulation Mode to capture and inspect network traffic generated by a web browser communicating with a web server.

The Packet Tracer Event List allows each protocol to be viewed independently while examining the encapsulation process across the OSI and TCP/IP models. :contentReference[oaicite:0]{index=0}
