## Overview

This walkthrough demonstrates the installation and physical connectivity of an enterprise router within a rack-mounted network infrastructure environment.

<!--more-->

The exercise focuses on hardware installation, fiber connectivity, WAN connectivity, power redundancy, and structured cabling practices.

---

---

## Step 1: Install the Router

Mount the router in the designated rack location.

Verify:

* Proper rack placement
* Secure mounting
* Accessibility of interfaces

---

## Step 2: Install Interface Modules

Install the required SFP transceivers.

Verify:

* Proper seating
* Correct interface type
* Secure installation

---

## Step 3: Connect Power

Connect the router to protected power infrastructure.

Verify:

* Power connection is secure
* UPS-backed power is available
* Device powers on successfully

---

## Step 4: Connect Fiber Uplinks

Connect the router interfaces to the fiber distribution system.

Verify:

* Proper transmit and receive alignment
* Correct port assignments
* Secure fiber connections

---

## Step 5: Connect WAN Connectivity

Connect the WAN interface to the upstream network device.

Verify:

* Correct cable type
* Proper interface selection
* Physical link status

---

## Step 6: Verify Physical Connectivity

Review all connections.

Confirm:

* Router is powered on
* Interfaces are connected
* Fiber links are seated correctly
* WAN connectivity is established
