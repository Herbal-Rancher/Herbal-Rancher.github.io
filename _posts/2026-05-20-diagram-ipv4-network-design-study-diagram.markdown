## Key Takeaways

- Begin every network by gathering business and technical requirements.
- Choose the smallest subnet that satisfies current host requirements while allowing room for growth.
- Use VLANs to logically separate departments, services, and security zones.
- Assign consistent gateway addresses and document the addressing plan.
- Validate connectivity before placing the network into production.
- Good network documentation simplifies future maintenance and troubleshooting.
