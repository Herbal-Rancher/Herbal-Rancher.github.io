## Overview

This no-narration walkthrough demonstrates how MAC spoofing works by using SMAC to change a workstation's MAC address and observe its effect on network communications.

## Lab Objectives

* Identify MAC addresses on multiple hosts
* Compare Ethernet adapter information
* Spoof a MAC address using SMAC
* Refresh DHCP addressing
* Verify changes to network identity
* Analyze the impact of MAC spoofing

This lab provides hands-on experience with a common security technique used in both legitimate testing and malicious network attacks.
