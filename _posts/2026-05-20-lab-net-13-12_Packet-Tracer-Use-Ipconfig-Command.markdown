## Overview

This guided technical walkthrough demonstrates how to use the **ipconfig /all** command to identify and correct an IPv4 addressing misconfiguration using Cisco Packet Tracer.

## Learning Objectives

By completing this exercise you will learn how to:

- Display network adapter configuration

- Verify subnet masks
- Verify default gateways
- Identify host configuration errors
- Correct addressing issues

## Why This Matters

The **ipconfig /all** command is one of the most commonly used troubleshooting tools for Windows systems. Quickly identifying incorrect IP addressing, subnet masks, gateways, or DNS settings is an essential skill for help desk technicians, network administrators, and cybersecurity professionals.
